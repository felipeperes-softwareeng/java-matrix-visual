# Java Matrix Visual 🧩

Implementação de uma matriz em Java com visualização criada no Miro. Projeto desenvolvido para ajudar estudantes a entenderem a estrutura de matrizes na JVM de forma visual e prática.

## 📚 Sobre o Projeto

Este projeto nasceu de um aprendizado prático: criar uma matriz visual no **Miro** e transformá-la em código Java, mapeando todos os valores de todas as colunas.

A ideia é simples — **ver antes de codar**. A visualização ajuda a entender a estrutura antes de partir para a implementação.
